# Aula de Pandas

Este repositorio foi criado como ponto de partida para as aulas de Pandas.
A ideia e que os alunos possam clonar o projeto e ja tenham acesso aos
datasets usados em sala, sem precisar baixar ou montar os arquivos manualmente.

## O que tem aqui

- `src/pandas/main.ipynb`: notebook inicial para acompanhar as aulas.
- `src/pandas/dataset/clientes.csv`: dataset de clientes.
- `src/pandas/dataset/vendas.csv`: dataset de vendas.
- `src/pandas/dataset/vendedores.csv`: dataset de vendedores.

## Como baixar o projeto

No terminal, rode:

```bash
git clone https://github.com/pedromaia-aluno/aula_pandas.git
cd aula_pandas
```

Depois disso, abra a pasta no VS Code, PyCharm, Jupyter, Google Colab ou na
ferramenta que estiver sendo usada na aula.

## Observacao para os alunos

Este projeto serve como material de apoio das aulas. Durante o curso, use os
datasets daqui para praticar leitura de arquivos, filtros, agrupamentos,
relacionamento entre tabelas e outras operacoes com Pandas.
